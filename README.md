# SnipeDesign (Phase 1)

Electron MVP for **Snipe** rig simulation (mast + spreaders + standing rigging), focused on **robust tension-only cables** and a stable nonlinear equilibrium solve.

Docs:

- `docs/PHASE1_TECH_SPEC.md`
- `docs/SNIPE_CONTROLS.md`
- `docs/PROMPT_FULL_ES.md`

## Run

1. Install deps: `npm install`
2. Start app: `npm start`

## Tests (solver)

`npm test`
